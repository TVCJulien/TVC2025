## Binaire

- ### Complétement à deux, méthode hyper facile

```bash
214 - 116

214 = 1 1 0 1 0 1 1 0

116 = 0 1 1 1 0 1 0 0

(Pour le complétement une fois le premier bit 1 rencontrer inversé tout les bits à droite)

complètement à 2 de 116 = 1 0 0 0 1 1 0 0 
```
